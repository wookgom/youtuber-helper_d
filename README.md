# youtuber-helper

## 시스템 정의서
### 1.  시스템 명칭

스트리머 및 편집자 지원 소프트웨어

### 2. 시스템 설명

스포츠 경기, 게임 방송 등 긴 영상 콘텐츠에서 자동으로 주요 장면을 추출하고 스트리밍 중 시청자 행동을 분석하는 AI 기반 방송 지원 시스템이다. 스트리머는 하이라이트 콘텐츠를 효율적으로 제작하고 시청자 집중도를 실시간으로 모니터링하여 콘텐츠 전략을 즉시 조정할 수 있다.

### 3. 운영 방식

이 시스템은 AI 기반의 웹 소프트웨어이다.

- 프론트엔드 (웹 브라우저): 스트리머가 보는 대시보드
    - 스트리머의 방송 화면과 소리를 WebRTC을 이용해 서버로 실시간 전송한다.
    - 분석이 완료된 결과 웹소켓을 통해 서버로부터 실시간으로 받아와 화면에  표시해준다.
- 백엔드
    - 웹 브라우저로부터 스트리밍 영상을 보낸다.
    - 멀티모달 처리가 가능한 모델을 통해 분석을 진행한다.
    - 분석이 끝난 데이터를 즉시 웹 브라우저로 다시 보내준다.

### 4. 핵심 사용자

- 유튜버, 스트리머 : 
긴 영상 콘텐츠를 자주 제작하는 개인 방송자이다.
- 영상 편집자 :
자동 하이라이트 생성 데이터를 활용해 효율적인 영상 편집을 희망하는 편집자이다.

### 5. 시스템 개발 목적

- 편집 시간 및 비용의 획기적 절감: 수동으로 하이라이트를 찾고 편집하는 데 드는 막대한 시간과 인력 비용을 자동화를 통해 해결한다.
- 콘텐츠 재가공 및 확산 용이성 증대: 원본 영상을 손쉽게 하이라이트, 쇼츠 등 다양한 형태의 콘텐츠로 재생산하여 채널 홍보 및 수익 창출 기회를 확대한다.
- 데이터 기반의 방송 운영 지원: 스트리머의 '감'에 의존하던 시청자 반응 파악을 데이터 기반의 예측 분석으로 전환하여, 방송 품질과 시청자 만족도를 과학적으로 향상시킨다.

### 6. 주요 기능

1. 실시간 시청자 반응 분석
2. 빠른 컷편집을 위한 AI 기반 하이라이트 타임라인 추천
3. 최종 스트리밍 분석 기능
4. 시청자 이탈 가능성 예측 및 위험 알림
5. 영상 내 흡연, 음주 등의 콘텐츠 안정성 분석 기능
